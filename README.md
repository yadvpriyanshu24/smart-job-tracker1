# 🚀 Smart Job Tracker (AI Powered)

A MERN stack job tracking platform that helps users manage job applications, track interview progress, upload resumes, and analyze resumes .

## 🌐 Live Demo

**Frontend (Live Website):**
https://smart-job-tracker1.vercel.app/

**Backend API:**
https://smart-job-tracker1.onrender.com

## 📌 Features

* 🔐 User Authentication (Register/Login with JWT)
* 📋 Track Job Applications
* ✏️ Add, Edit, Delete Jobs
* 📊 Track Application Status
* 👤 User Profile Management
* 📄 Resume Upload Feature
* 🤖 Resume Analysis
* 📱 Responsive UI
* ☁️ Full MERN Deployment (Vercel + Render)

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Multer

### Deployment

* Vercel (Frontend)
* Render (Backend)
* MongoDB Atlas (Database)



## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/yadvpriyanshu24/smart-job-tracker1.git
```

### Backend Setup

```bash
cd server
npm install
npm start
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

## 🔑 Environment Variables

Create `.env` file in client:

```env
VITE_API_URL=http://localhost:5000
```

Backend `.env`:

```env
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
PORT=5000
```
